# Backyard Survivor Wiki

A dark, tribal-themed wiki for tracking Backyard Survivor seasons — cast pages, voting history, challenges, and elimination order.

## Quick Start

```bash
npm install
npm run dev
```

## Updating Season Data

Edit `src/data.js` — that's the only file you need to touch. Follow the existing structure to add players, votes, and challenges.

## Deploy

Push to `main` and GitHub Actions will auto-deploy to GitHub Pages.
